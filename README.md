# AppScreenShotsControl

программа прензначена для отслеживания действий пользователя на пк
путем снятия скриншотов экрана через определенные промежутки времени
и отправки скриншотов на ресурс Dropbox

программа создана в учебных целях для взаимодействия со сторонними API
в данном случае Dropbox, и не предназначенна ни для каких бы то ни было
шпионских движух
